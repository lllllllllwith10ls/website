Category 5: Pentagonal Rectates

This deals with honeycombs with pentagonal symmetric verfs. There are 19 honeycombs, with three groups of six and a lone operative.

Ripech regiment

This regiment has doehonnic symmetry.

123. Ripech - (RIP ech) rectified pechon. It has coes and ikes.

124. Rifpech - (RIFF pech) rectified fipech. It has tepeats and gads.

125. Papviddid - (Pap VID id) pentagramal antiprismatoverted disdoe. It has gads and tis.

126. Firpech - (FIRP ech) facetorectified pechon. It has tepeats and tis.

127. Nipcid Tittip - (NIP sid TIT ip) interceptipental cube-doe-3-3-5. It has coes, gads, and sitpetrats.

128. Omfapid Petittip - (om FAP id pet IT ip) omnicircumfacialpental doe-peat-3-3-5. It has ikes, tepeats, and sitpetrats.

Ripped regiment

This regiment has pedhonnic symmetry.

129. Ripped - (RIP ed) rectified pedhon. It has ids and ikes.

130. Rifped - (RIFF ped) rectified fipped. It has peats and gads.

131. Papvidipep - (Pap VID ip pep) pentagramal antiprismatoverted doe-pepat. It has gads and topepats.

132. Firped - (FIRP ed) facetorectified pedhon. It has peats and topepats.

133. Nipdid Pattip - (NIP did PAT ip) interceptipental disdoe-5-3-5. It has ids, gads, and sitpepats.

134. Omfapid Pepaptip - (om FAP id pep AP tip) omnicircumfacialpental doe-pepat-5-3-5. It has ids, gads, and sitpepats.

Riphexah regiment

This regiment has phexahic symmetry.

135. Riphexah - (RIFF ex ah) rectified phexah. It has thats and ikes.

136. Rifiphexah - (RIFF iff ex ah) rectified fiphexah. It has phats and gads.

137. Papvihadid - (PAP vi dap) pentagramal antiprismatoverted doe-azpat. It has gads and topazats.

138. Firphexah - (FIRF ex ah) facetorectified phexah. It has phats and topazats.

139. Niphid Aztap - (NI fid AZ tap) interceptipental hexat-doe-inf-3-5. It has thats, gads, and sitpazts.

140. Omfapidap Aztap - (om FAP id ap AZ tap) omnicircumfacialpental doe-azpat-inf-3-5. It has ikes, phats, and sitpazts.

Risped regiment

This lone operative has pedhonnic symmetry.

141. Risped - (RIS ped) rectified sipped. It has dids and pepats.
